# 제네시스 코어: DNA 및 유전 알고리즘(GA) 설계

## 1. 개요
DNA는 개체의 신체, 성향, 뇌 구조를 결정하는 데이터 스트림입니다. 스크립트 없이 환경 적응을 구현하기 위해 철저하게 유전 알고리즘에 기반하여 진화합니다.

## 2. DNA의 구조 (Genome Structure)
DNA는 Float 배열 (0.0 ~ 1.0 정규화)로 구성된 염색체 집합입니다.

### 2.1. 신체 염색체 (Physical Genes)
*   `Size (크기):` 방어력과 공격력 증가, 대사량(에너지 소모) 증가.
*   `Speed (속도):` 이동 속도, 굶주림 속도 증가.
*   `Digestive_Enzyme (소화 효소):` [식물 소화력, 고기 소화력]. (1.0, 0.0)이면 완벽한 초식, (0.5, 0.5)면 잡식.
*   `Fur_Density (털 밀도):` 추위 저항 증가, 더위 저항 감소.
*   `Lifespan (수명):` 최대 수명 사이클.

### 2.2. 심리/본능 염색체 (Psychological Genes)
*   `Aggression (공격성):` 타 개체 조우 시 공격 행동을 선택할 가중치.
*   `Sociality (사회성):` 동족과 군집을 이루려는 성향 거리.
*   `Curiosity (호기심):` 미지의 오브젝트와 상호작용할 확률 (기술 발전 트리거).

### 2.3. 후성 유전 (Epigenetic Marks)
*   경험을 통해 후천적으로 메틸화(변형)되는 데이터 공간.
*   `Fear_Vector:` 반복적으로 피해를 입은 특정 형태(적의 모습 데이터)를 저장하여, 다음 세대가 해당 형태에 즉각적인 공포(회피 가중치)를 가지게 함.

## 3. 진화 프로세스 (Evolutionary Cycle)

### 3.1. 적합도 평가 (Fitness Evaluation)
인위적인 적합도 함수는 없습니다. "생존하여 번식했다면 적합한 것"입니다. 수명이 다하거나 굶어 죽은 개체는 유전자를 남기지 못하고 도태됩니다.

### 3.2. 교차 연산 (Crossover)
생식(Mating) 시 두 부모의 DNA 배열을 섞습니다.
*   **균등 교차 (Uniform Crossover):** 부모 A와 B의 각 유전자를 50% 확률로 무작위로 물려받음.

### 3.3. 돌연변이 (Mutation)
세대 교체 시 일정한 확률로 유전자 값이 랜덤하게 변경됩니다.
*   **기본 돌연변이율:** 약 1~5%.
*   **동적 돌연변이:** 개체가 살아생전 극한의 환경 스트레스(방사능, 기아 상태 지속, 극심한 온도)를 받으면, 체내의 '안정성' 수치가 떨어져 자손의 돌연변이율이 최대 20%까지 폭증합니다. (급격한 진화적 도약, 즉 단속평형설 구현).

## 4. 종의 분화 (Speciation)
두 개체 간의 DNA 거리(유클리디안 거리 등)를 측정합니다.
*   이 거리가 `Speciation Threshold` (종 분화 임계값)을 초과하면 두 개체는 서로 생식 코드가 맞지 않아 번식할 수 없습니다.
*   이때 시스템은 새로운 종(Species ID)을 할당하고 데이터베이스에 기록합니다. (예: 종 1에서 종 2 분화).
