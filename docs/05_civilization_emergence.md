# 제네시스 코어: 창발적 문명 및 기술(Emergent Civilization & Tech) 설계

## 1. 개요
문명의 발전 단계(석기 -> 청동기 -> 철기)는 스크립트로 강제되지 않습니다. 물리 엔진과 객체 간의 상호작용 매트릭스 속에서 우연한 발견과 학습의 축적으로 기술 트리가 '스스로 완성'됩니다.

## 2. 물리적 속성과 상호작용 매트릭스

### 2.1. 아이템/사물 컴포넌트 (Material Component)
세계에 존재하는 모든 객체(돌, 나무, 광석, 고기)는 아래와 같은 물리적 속성을 갖습니다.
*   `Hardness (경도):` 충돌 시 파괴 여부 결정.
*   `Flammability (가연성):` 불이 붙을 확률 및 타는 시간.
*   `Nutrition (영양):` 섭취 시 회복되는 체력/허기.
*   `Sharpness (날카로움):` 무기나 도구로 사용 시 데미지 계수.

### 2.2. 상호작용 조합 (Crafting by Physics)
플레이어가 레시피를 주는 것이 아닙니다. 개체의 무작위 행동(A와 B를 부딪히기, A를 B에 던지기)을 물리 엔진이 연산하여 결과를 도출합니다.
*   [경도 높은 돌] + [경도 높은 돌] 타격 -> 스파크 발생 파티클.
*   [스파크] + [가연성 높은 마른 풀] -> [불 (Fire Entity)] 생성.
*   [불] + [날고기] -> [구워진 고기 (소화 효율, 영양분 극대화, 부패 지연)].
*   [불] + [특정 광석] -> 광석의 내구도가 깎이며 [액체 금속]으로 상태 변화 -> 식으면 [금속 도구] 생성.

## 3. 기술의 확산 (Imitation Learning & Culture)

### 3.1. 유레카 틱 (Eureka Moment)
호기심 DNA 수치가 높은 돌연변이 개체가 우연히 [돌]로 [나무]를 쳐서 [장작]을 얻고 이를 불태워 추위를 이겨내면 엄청난 '생존 보상(Reward)'이 뇌신경망에 주어집니다.

### 3.2. 문화의 전파 (Cultural Transmission)
사회성 수치가 높은 무리 내에서는 '모방'이 일어납니다. 보상을 얻은 행동 패턴은 시각적 관찰을 통해 다른 개체의 신경망으로 가중치가 복사(학습)됩니다. 이것이 '기술의 보급'입니다.
*   해당 기술(도구 사용)을 익힌 무리는 생존률이 압도적으로 높아지며 문명(국가)의 형태를 갖추게 됩니다.

## 4. 사회 구조의 창발

### 4.1. 역할 분담과 계급
*   **잉여 자원:** 농경(씨앗을 땅에 두었더니 자라는 현상 학습)으로 식량이 남기 시작하면, 모든 개체가 사냥/채집을 할 필요가 없어집니다.
*   **분업화:** 힘이 센 DNA를 가진 개체 군은 무력을 행사해 식량을 빼앗거나 지키는 '군인/지배층'이 되고, 지능이 높은 객체는 '샤먼/기술자'가 되는 등 자연스러운 계급화가 발생합니다.

### 4.2. 경제 메커니즘
특정 지역에 부족한 자원(예: 소금, 특수 광석)에 대한 개체들의 '욕구 수치(Demand)'가 올라가면, 잉여 자원을 들고 이동하다 서로 만나 아이템을 교환(Drop & Pick)하는 행동이 보상으로 강화되며 '무역'이 발생합니다.
