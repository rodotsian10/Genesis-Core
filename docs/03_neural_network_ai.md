# 제네시스 코어: 생명체 뇌신경망(Neural Network) 인공지능 설계

## 1. 개요
개체는 상태 머신(FSM)이나 행동 트리(Behavior Tree) 같은 하드코딩된 AI를 사용하지 않습니다. 모든 개체는 다층 퍼셉트론(MLP) 또는 NEAT(NeuroEvolution of Augmenting Topologies) 알고리즘 기반의 뇌(신경망)를 가집니다.

## 2. NEAT 알고리즘 적용 (신경망 자체의 진화)
단순히 가중치만 진화하는 것이 아니라, 뇌 신경망의 **구조 자체(노드와 은닉층의 개수)** 가 진화합니다. 
*   초기 생명체의 뇌는 입력층과 출력층이 직접 연결된 단순 구조입니다.
*   시간이 지나며 DNA 돌연변이를 통해 새로운 '뉴런(은닉 노드)'이 추가되며 복잡한 사고(조건부 판단, 기억 연산)가 가능해집니다.

## 3. 신경망 입출력 아키텍처

### 3.1. 입력층 (Input Layer / Senses)
개체가 인지하는 세상의 상태를 정규화된 값(-1.0 ~ 1.0)으로 입력받습니다.
*   **내부 상태:** `Health` (체력), `Hunger` (허기), `Thirst` (목마름), `Fear_Level` (현재 공포 수치), `Lust` (번식욕).
*   **시각 (Vision Raycast):** 전방 N개의 Ray를 발사하여 닿은 물체의 데이터 입력.
    *   `Distance_to_Object` (물체까지의 거리)
    *   `Type_of_Object` (0.1=물, 0.5=식물, 1.0=다른 생명체)
    *   `Size_of_Object` (물체의 크기)
*   **청각 (Hearing):** 반경 내 가장 큰 소리의 방향 벡터와 강도.

### 3.2. 은닉층 (Hidden Layer / Brain Processing)
*   DNA에 의해 결정된 수의 뉴런과 시냅스 구조를 가집니다.
*   활성화 함수로 Tanh 또는 ReLU를 사용하여 비선형적인 결정을 내립니다.

### 3.3. 출력층 (Output Layer / Actions)
신경망 연산 결과 도출된 가장 값이 높은 노드의 행동을 실행합니다.
*   `Move_Forward`, `Turn_Left`, `Turn_Right` (이동 제어)
*   `Eat` (입 앞의 대상 섭취 시도)
*   `Attack` (앞의 대상 공격)
*   `Mate` (번식 시도)
*   `Grab/Drop` (도구/사물 쥐기 및 놓기)
*   `Vocalize` (소리 내기 - 언어 및 정보 전달의 기초)

## 4. 학습 메커니즘 (Learning & Reinforcement)
*   **유전적 뇌 (선천적):** 태어날 때 부모의 신경망 가중치를 물려받아 기초적인 행동(본능)이 가능합니다.
*   **강화학습 (후천적):** 살아가는 동안 행동에 따른 보상(허기 감소, 체력 회복)과 처벌(데미지 입음, 통증)을 받아 뇌의 가중치를 실시간 업데이트(역전파 또는 로컬 Hebbian Learning)하여 경험적 학습을 이룹니다.
